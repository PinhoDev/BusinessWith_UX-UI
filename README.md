# SystemGuide Questionnaire

Online questionnaire for BusinessWith SystemGuide UX research study.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Build for production:
```bash
npm run build
```

## Features

- 6 simple questions based on UX research
- Progress tracking with visual indicator
- Time estimate display
- Responsive design
- Clean, modern UI
- Anonymous responses
- Results summary

## Tech Stack

- React 18
- Vite
- CSS3

## Usage

Share the live URL with participants to collect responses for the UX case study.
